Щебетарь (ICQ Messenger)
Щебетарь — це клієнт-серверна система обміну миттєвими повідомленнями (месенджер), розроблена на мові Java з використанням бібліотеки JavaFX. 
Проєкт реалізує повноцінну комунікацію в реальному часі, передачу файлів та систему керування профілями.

1. Технологічний стек
Мова: Java

Інтерфейс: JavaFX (MVC-архітектура)

Мережа: Sockets (TCP/IP)

Серіалізація: Власний парсинг XML-повідомлень

Кодування: Base64 (для передачі файлів та аватарок)

Багатопотоковість: Java Concurrency (сервер обробляє кожного клієнта в окремому потоці)

3. Основні можливості

Авторизація: Підключення до сервера за IP-адресою та портом.

Глобальний та приватні чати: Можливість спілкуватися як з усіма користувачами одночасно, так і в приватних діалогах.

Кастомізація: Зміна імені користувача та встановлення власної аватарки (зображення автоматично синхронізуються між усіма клієнтами).

Передача файлів: Обмін документами та зображеннями з функцією збереження.

Система сповіщень: Індикація непрочитаних повідомлень у списку контактів.

Системні повідомлення: Сповіщення про вхід/вихід користувачів та зміну їхніх даних.

Зручність: Копіювання тексту повідомлень через контекстне меню (ПКМ).

4. Архітектура проєкту

Проєкт розділений на дві логічні частини:

Server: Приймає з'єднання, маршрутизує повідомлення між клієнтами, моніторить активність та зберігає стан (аватарки, список підключень).

Client: Графічний інтерфейс користувача, який забезпечує взаємодію з сервером та локальне зберігання історії чатів.

5. Як запустити
Запуск сервера: Запустіть клас com.messenger.server.ServerApp.

Запуск клієнта: Запустіть клас com.messenger.client.ClientApp.

При вході введіть IP-адресу (наприклад, localhost) та порт 3128.
